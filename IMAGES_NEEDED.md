## How to Add Your Images

1. **Convert HEIC to JPG** (if needed): On Mac, you can open HEIC files in Preview → File → Export and choose JPEG.
2. **Save each image** into the `images/` folder using the exact filenames below.
3. Supported formats: `.jpg`, `.jpeg`, or `.png` (rename to `.jpg` in the list if you use PNG).
